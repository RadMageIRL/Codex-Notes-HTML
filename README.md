# Codex-Notes-HTML
A lightweight, single-file, local-first note-taking application designed for speed, privacy, and simplicity. Codex runs entirely in your browser without requiring a backend, server, or cloud account.

### Key Features

*   **Single-File Architecture**: Everything (HTML, CSS, JS) is contained within one file. No installation or complex setup required[cite: 1].
*   **Local-First Storage**: Uses Browser IndexedDB for data persistence and offers dual-write backups for data safety[cite: 1].
*   **Developer-Friendly**: Includes syntax highlighting (via Prism.js), a clean monospaced aesthetic, and features like line numbers[cite: 1].
*   **Organized**: Robust system for folders, tags, and pinned notes[cite: 1].
*   **Feature-Rich Editor**:
    *   Find/Replace functionality[cite: 1].
    *   Version history with "diff" views to see changes[cite: 1].
    *   Print to PDF and JSON export/import support[cite: 1].
    *   Checklist functionality[cite: 1].
*   **Visual Customization**: Supports light/dark themes and customizable folder color labels[cite: 1].
